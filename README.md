# FF14 零式タイムライン管理ツール - 実行方法

## プロジェクト概要
FF14の高難易度コンテンツ（零式レイド）のタイムライン管理ツールです。Vue.js 3 + Vite + Tailwind CSSで構築されています。

## 前提条件
- Node.js（推奨: v18以上）
- npm または yarn

## 実行手順

### 1. 実装ディレクトリへ移動
```bash
cd 04_implementation
```

### 2. 依存パッケージのインストール（初回のみ）
```bash
npm install
```

### 3. 開発サーバーの起動
```bash
npm run dev
```

実行後、ターミナルに表示されるローカルサーバーURL（通常は `http://localhost:5173`）にブラウザでアクセスしてください。

### 4. プロダクションビルド（本番環境用）
```bash
npm run build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。

### 5. ビルド結果のプレビュー
```bash
npm run preview
```

ビルドされたアプリケーションをローカルでプレビューできます。

## 主な機能
- ボスのギミックタイムラインの視覚的管理
- パーティメンバー8人分のジョブ・スキル配置
- フェーズ管理（追加・編集・削除）
- ドラッグ&ドロップによる直感的な操作
- タイムラインパターンの保存・読込
- URL共有機能

## 技術スタック
| 項目 | 技術 |
|------|------|
| フレームワーク | Vue.js 3 (Composition API) |
| 状態管理 | Pinia |
| スタイリング | Tailwind CSS |
| ビルドツール | Vite |
| 言語 | TypeScript |

## ディレクトリ構造
```
04_implementation/
├── src/
│   ├── components/     # Vueコンポーネント
│   ├── composables/    # Composition API関数
│   ├── stores/         # Pinia状態管理
│   ├── types/          # TypeScript型定義
│   ├── utils/          # ユーティリティ関数
│   └── styles/         # スタイルシート
├── public/             # 静的ファイル
└── dist/               # ビルド出力先
```
